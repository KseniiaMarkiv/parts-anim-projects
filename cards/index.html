Image splitter

Image splitter is a free online tool, where you can split images in to pieces in all directions (vertical, horizontal or both). Splitter is added in image, user can drag and place the splitter in the position, where they want to split the image exactly. Split image in to piece option is used to split images to matrix or tile based by providing rows and columns number.

How this tool working?

Image splitter tool split image using two options, First one is loads the draggable splitter along with image and second one using matrix type splitter by providing number of rows and columns as input. Once set, tool will read the co-ordinates and start process to break the image. Finally, preview of split image is displayed along with download button.

Is it free tool?

Image splitter tool is completely free to use and it is a full version, no hidden payments, no signup required, no demo versions and no other limitations.You can split any number of images without any restriction.

Are my images secured?

Yes, Split tool break the image in browser, No worry, your images are not uploaded to server or anywhere else, so its highly secured and no one can access your images except you.

Download option

Its is easy to download split images all at once in zipped format, once image is cut in to pieces, preview of split images is displayed in tool along with download button in zipped format.
